# Docs Index

This folder is organized by purpose, not by chronology.

## Structure

- `docs/project/`
  Current project state, MVP test paths, and execution context.
- `docs/product/`
  Product philosophy, UX framework, and progressive disclosure model.
- `docs/drill/`
  Drill-engine and knowledge-graph invariants, plus manual drill eval docs.
- `docs/codex/`
  Agent, session, and Codex workflow documentation.
- `docs/theta/`
  Research-track state for evidence-backed product claims.
- `docs/reference/`
  Reference artifacts and example outputs.
