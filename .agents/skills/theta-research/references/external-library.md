## Usage Rules

- Do not copy the full PDF library into this repo by default.
- Read only the subset relevant to the active question.
- Convert useful papers into repo-native notes under `references/`.
- Keep the raw PDFs in the external library unless there is a clear reason to vendor a file.

## Theta Workflow

1. Start with `00-index.md`.
2. Check this file to find the external library root.
3. Select only the papers relevant to the current question or phase.
4. Write structured notes into the repo phase docs.
5. Distinguish direct evidence, indirect evidence, and speculation.
