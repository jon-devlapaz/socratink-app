# Paper Notes Index

This folder holds one reusable markdown note per paper.

## Purpose

Use `paper-notes/` for durable, paper-specific notes.

Use the phase docs for:

- synthesis
- cross-paper comparisons
- phase-level product implications

## Naming Convention

Use short, stable filenames:

- `author-year-keyword.md`

Examples:

- `dunlosky-2013-learning-techniques.md`
- `norman-2013-retrieval-competition.md`
- `wing-marsh-cabeza-2013-reinstatement.md`

## Minimum Contents

Each paper note should be based on `assets/evidence-template.md` and include:

- citation
- relevance to phase(s)
- direct vs indirect evidence
- limits
- safe vs risky product implications

## Workflow

1. Screen a paper with `assets/evidence-screen-template.md`.
2. If it is worth deeper reading, create a file in `paper-notes/`.
3. Add the paper note to this index.
4. Propagate only the reusable synthesis into the phase docs.
