# Theta Research Index

Use this directory to organize the evidence base behind the product's learning claims.

## Reference Flow

1. Read this index first.
2. Read `external-library.md` if the needed papers live outside the repo.
3. Use `paper-notes/00-index.md` when creating or locating durable paper-specific notes.
4. Move to the relevant phase document for synthesis and product implications.
5. Store source PDFs or exported notes under `references/raw/` only when appropriate.

## Phase Files

- `phase-1.md`
  Entry phase. Attention, orientation, curiosity, and initial encoding.
- `phase-2.md`
  Active reconstruction, retrieval, and explanation from memory.
- `phase-3.md`
  Feedback, gap detection, correction, and misconception repair.
- `phase-4.md`
  Consolidation, spacing, durable retention, and transfer.

## Other Reference Layers

- `external-library.md`
  Pointer to the raw external PDF collection.
- `paper-notes/00-index.md`
  Index of durable paper-by-paper markdown notes.

## Evidence Standard

- Prefer meta-analyses, reviews, and primary studies over blog summaries.
- Mark each claim as:
  - direct evidence
  - indirect evidence
  - hypothesis
- Note when product language goes beyond the literature.
